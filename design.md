# Vision

Create the highest-end detailing website in Southern Minnesota.

The website should feel less like a local detailing business and more like a premium automotive brand.

Users should immediately associate Mecha Auto Spa with:

- Precision
- Luxury
- Craftsmanship
- Trust
- Professionalism
- Convenience
- Technology

The website should rival companies like Porsche, Rivian, Vercel, Apple, and modern SaaS landing pages while remaining approachable for local customers.

---

# Brand Personality

The company is not a "car wash."

It is an automotive appearance specialist.

Tone:

- Professional
- Premium
- Technical
- Honest
- Minimal
- Clean
- Confident

Avoid:

- Loud marketing
- Flashing graphics
- Carbon fiber overload
- Fake chrome
- Cheap gradients
- Overly aggressive typography

The experience should feel expensive.

---

# Design Philosophy

Every page should answer one question:

"Would someone trust this company with a $120,000 vehicle?"

If the answer is yes—

the design is successful.

# Core Concept

The services section is not a pricing list.

It is a **premium product catalog system**.

Each service behaves like a landing page for a high-end product.

Every page must:

- Sell visually first
- Explain second
- Price third
- Convert always

# Animation System

## Rules

- Motion must feel expensive
- Subtle > flashy

---

## Standard Animations

### Fade Up

- duration: 0.6s
- easing: easeOut
- stagger children

---

### Page Transition

- opacity: 0 → 1
- slight y offset
- 0.4s duration

Cards:

- translateY(-4px)

- border highlight

- glow accent
